# Referência funcional do sistema original

Esta pasta contém o executável do sistema legado e suas bibliotecas, exclusivamente para consulta de comportamento e telas. Ela não contém configuração de banco e, portanto, não deve ser executada nem configurada pela Lovable.

Para implementar as funções no CLIM, use o inventário em `../03-DOCUMENTACAO/LEGACY-FUNCTIONAL-PARITY.md`. O executável é apenas uma referência visual/funcional do que cada tela e botão fazia antes da modernização.

O arquivo `config.xml`, backups, instaladores e configuradores foram removidos antes da exportação por conterem ou poderem reconstruir informações privadas da operação.
