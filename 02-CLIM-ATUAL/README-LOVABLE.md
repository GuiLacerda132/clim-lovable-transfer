# Base de desenvolvimento CLIM

Este diretório é uma cópia segura do código atual do CLIM. Ele já possui:

- aplicativo desktop Electron;
- React/TanStack Start;
- design CLIM;
- repositório PostgreSQL e contratos tipados do legado;
- paginação otimizada de pacientes;
- rotas e componentes existentes.

Não há credenciais, banco, backup ou dados de pacientes. A configuração privada deve permanecer fora do GitHub. Para o objetivo de implementação, leia primeiro `../03-DOCUMENTACAO/LOVABLE-IMPLEMENTATION-BRIEF.md`.
